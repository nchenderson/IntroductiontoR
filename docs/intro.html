1.1 Using R as a calculator

You can use R as a basic calculator.

For example:

42 + 17 
## [1] 59
sqrt(243)
## [1] 15.58846
1.56*1233
## [1] 1923.48
7.21*8^4
## [1] 29532.16

For more complicated mathematical operations, it is useful to store intermediate values in named variables.

x <- (42 + 17)*sqrt(43)
y <- 7.21*8^4 + log(2.34)
z <- x/y
z  ## print out the value of z
## [1] 0.01310022

Here, x, y, and z are examples of variables.

The pair of characters <- used together is known as the assignment operator in R. x <- 2 assigns the value 2 to the variable x.
